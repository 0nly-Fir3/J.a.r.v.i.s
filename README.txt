JARVIS PC Controller v2

How to run:
1. Extract the ZIP.
2. Double-click install_requirements.bat.
3. Double-click start_jarvis.bat and keep it open.
4. Open jarvis_pc_v2.html or double-click open_jarvis_ui.bat.

Test commands:
- open Discord
- show system stats
- open downloads folder
- take screenshot
- press alt tab
- press windows d
- copy
- paste
- read clipboard
- type hello world
- set volume to 30
- close Discord
- confirm

Teach app:
remember app blender is C:\Program Files\Blender Foundation\Blender 4.3\blender.exe

Safety:
Deletes, formatting, registry editing, arbitrary CMD/PowerShell, and antivirus disabling are blocked.
