JARVIS V10 - Personal OS Layer
================================

This is the V10 build: OpenRouter brain + local Windows control + screen reading + task planner + routines + projects + file brain + optional Chrome extension bridge.

QUICK START
1. Extract the folder.
2. Run start.bat.
3. Add OPENROUTER_API_KEY to .env.
4. Open http://127.0.0.1:5050
5. For browser superpowers, load the Chrome extension:
   - Open Chrome
   - Go to chrome://extensions
   - Turn on Developer mode
   - Click Load unpacked
   - Select the chrome_extension folder inside this JARVIS folder
